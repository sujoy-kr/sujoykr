Group Chat

Group Chat - Connect with your friends faster.

Type of project

Full Stack

Technologies

Node.js

Express.js

Socket.IO

Moment

Year

2021

This was my first Socket.IO project. Users can type an username and select their preferred group to join and start chatting.

After joining a group, users can send messages to other users in the same group. The side panel shows the list of active users in the group. The welcome bot will greet the user when they join a group and will also send a message to other users. The bot will also send a message when an user leaves. The app also tracks messages time using the moment package in backend. This all happens in real time using Socket.IO.
