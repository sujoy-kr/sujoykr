Map Veto

Map Veto - A map veto system for Valorant tournaments.

Type of project

Frontend

Technologies

HTML

CSS

Year

2022

I created this project for Valorant tournaments. This app is a map veto system for Valorant tournaments where both teams get to choose their name, logo, theme color, number of matches, and maps.

After submitting the form, the teams will be redirected to a page where they can see their respective map veto. The team that won the previous match will get to choose the map first. The team that lost the previous match will get to choose the map second and so on. When a team bans a map, the color of the next cross will be changed to the other team's theme color. Banning will be disabled when the number of remaining maps is equal to the number of matches.
